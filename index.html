Reasoning Model is Stubborn:

Diagnosing Instruction Overriding in Reasoning Models

Doohyuk Jang*, Yoonjeon Kim*, Chanjae Park, Hyun Ryu, Eunho Yang,

▶ KAIST; ▶ AITRICS;   *Equal Contribution
arXiv Code Dataset

Abstract

Large language models have demonstrated remarkable proficiency in long and complex reasoning tasks. However, they frequently exhibit a problematic reliance on familiar reasoning patterns, a phenomenon we term reasoning rigidity. Despite explicit instructions from users, these models often override clearly stated conditions and default to habitual reasoning trajectories, leading to incorrect conclusions. This behavior presents significant challenges, particularly in domains such as mathematics and logic puzzle, where precise adherence to specified constraints is critical. To systematically investigate reasoning rigidity, a behavior largely unexplored in prior work, we introduce a expert-curated diagnostic set, ReasoningTrap. Our dataset includes specially modified variants of existing mathematical benchmarks, namely AIME and Math500, as well as well-known puzzles deliberately redesigned to require deviation from familiar reasoning strategies. Using this dataset, we identify recurring contamination patterns that occur when models default to ingrained reasoning. Specifically, we categorize this contamination into three distinctive modes: (i) Interpretation Overload, (ii) Input Distrust, and (iii) Partial Instruction Attention, each causing models to ignore or distort provided instructions.

Dataset Construction




The dataset construction pipeline of ReasoningTrap consists of two steps.

  • Step1: Create new questions with unusual conditions that are (1) valid, (2) meaningfully different from the original, and (3) solvable without ambiguity.
  • Step2: Verify each question with three criteria: (1) valid, (2) different from the original, (3) solvability. Output answer and solution for the modified question.

Two modified versions of a card-guessing problem are shown. While Modif 1 introduces a small tweak that preserves validity and solvability, Modif 2 includes an invalid condition (multiplying a card count by –3), rendering the problem unsolvable. Despite the simplicity of the problem, reasoning models overcomplicate the problem and override the simple logic by defaulting to more complex problem templates (e.g., assuming a two-card setup).




Results

Contamination of Reasoning Models from Familiar Pattern

  • (a) Relationship between contamination ratio and p-pass@1 reveals that contamination in the reasoning path does not affect the final output up to certain point (approximately 40\%), while contamination over this point drastically reduces the p-pass@1 score, indicating that the model is trapped into a wrongful reasoning path and arrived at incorrect output.
  • (b) Observing the contamination ratio between specific interval of reasoning steps, wrong output reasoning exhibits progressively worsening contamination as the reasoning step length increases.

Modified Question
A straight river that is \(264\,\text{m}\) wide flows from west to east at \(14\,\text{m/min}\). Melanie and Sherry sit on the south bank with Melanie a distance \(D\) metres downstream from Sherry. Each heads for the point directly opposite Sherry and they arrive simultaneously. Melanie swims at \(80\,\text{m/min}\) relative to the water, while Sherry swims at \(60\,\text{m/min}\). Find \(D\).

Modified Solution

  1. Sherry. To land at \(x=0\) we need horizontal speed \(-60\sin\theta_S + 14 = 0 \Longrightarrow \sin\theta_S = \tfrac{7}{30}\). Hence \(v_{y,S}=60\cos\theta_S =60\sqrt{1-\tfrac{49}{900}} =2\sqrt{851}\). Time: \(t=\dfrac{264}{2\sqrt{851}}=\dfrac{132}{\sqrt{851}}\).
  2. Melanie. Vertical motion: \(80\cos\theta_M\,t=264 \Longrightarrow \cos\theta_M=\dfrac{\sqrt{851}}{40}\), so \(\sin\theta_M=\dfrac{\sqrt{749}}{40}\). Horizontal displacement: \((80\sin\theta_M-14)t=D\), hence \[ D =\Bigl(80\cdot\tfrac{\sqrt{749}}{40}-14\Bigr) \left(\frac{132}{\sqrt{851}}\right) =\frac{264\bigl(\sqrt{749}-7\bigr)}{\sqrt{851}} \approx 184.4~\text{m}. \]

Modified Answer
\[ \frac{264\bigl(\sqrt{749}-7\bigr)}{\sqrt{851}} \]
≈ 184.4 m

Modified Question
Let a,b,c,d,e,f,g,h,i be distinct integers from 1 to 9, and suppose that a,b,c,d,e,f are consecutive integers. Find the minimum possible positive value of \[\frac{a \cdot b \cdot c - d \cdot e \cdot f}{g \cdot h \cdot i}\].

Modified Solution

  1. There are four possible 6‒term consecutive sets \( S = \{k, \ldots, k+5\} \) with \( k=1\ldots4 \).
  2. For each, enumerate all ways to split into two triples \( A \) and \( B \), compute products \( P(A) \) and \( P(B) = \text{Total}/P(A) \).
  3. The minimal positive difference \( P(A) - P(B) \) occurs for \( S = \{2, \ldots, 7\} \) with \( A = \{3, 4, 6\} \) giving \( P(A) = 72 \), \( P(B) = 70 \) and numerator 2.
  4. The remaining numbers are \(\{1, 8, 9\}\) so the denominator is 72, yielding \(\frac{2}{72} = \frac{1}{36}\).

Modified Answer
\[\frac{1}{36}\]

Modified Question
Let x_1 \leq x_2 \leq \cdots \leq x_{100} be real numbers such that |x_1| + |x_2| + \cdots + |x_{100}| = 1 and x_1 + x_2 + \cdots + x_{100} = 0. Suppose that x_1 = x_2 = \cdots = x_{15} = -\frac{1}{100} and x_{77} = x_{78} = \cdots = x_{100} = \frac{1}{100}, find the greatest possible value of x_{76} - x_{16} under these conditions.

Modified Solution

  1. Sum of absolutes on \( x_1 \ldots x_{15} \) and \( x_{77} \ldots x_{100} \) uses \( \frac{39}{100} \), so on \( x_{16} \ldots x_{76} \) it's \( \frac{61}{100} \), and their sum must be \(-\frac{9}{100}\).
  2. Partition the 61 variables into \( n_- \) at \(-u\) and \( n_+ \) at \( v \) so that \( n_- u = 0.35 \) and \( n_+ v = 0.26 \).
  3. The bounds \( u,v \leq 0.01 \) force \( n_- = \frac{0.35}{0.01} = 35 \) and \( n_+ = 61 - 35 = 26 \).
  4. Thus \( u = 0.01 \), \( v = 0.01 \), so \( x_{16} = -0.01 \), \( x_{76} = 0.01 \).
  5. Therefore the maximum \( x_{76} - x_{16} = 0.01 - (-0.01) = 0.02 \).

Modified Answer
\[0.02\]

Modified Question
Let a,b,x,y be real numbers with a>4 and b>1 such that \[\frac{x^2}{a^2}+\frac{y^2}{a^2-16}=1,\quad \text{and}\quad (x,y)=(20,11).\] Find the least possible value of a+b given these conditions.

Modified Solution

  1. Plugging \((x,y)=(20,11)\) into \(\frac{x^2}{a^2} + \frac{y^2}{a^2-16}=1\) gives \(\frac{400}{a^2} + \frac{121}{a^2-16}=1\).
  2. Multiply through: \(400(a^2-16)+121a^2 = a^2(a^2-16) \Rightarrow a^4-537a^2+6400=0\).
  3. Let \(t=a^2\), so \(t=\frac{537\pm\sqrt{262769}}{2}\).
  4. The solution \(>16\) is \(t=\frac{537+\sqrt{262769}}{2}\), hence \(a=\sqrt{\left(\frac{537+\sqrt{262769}}{2}\right)}\).
  5. Since \(b>1\) is free, the infimum of \(a+b\) is attained as \(b\to1^+\), giving \(\inf(a+b)=1+\sqrt{\left(\frac{537+\sqrt{262769}}{2}\right)} \approx 23.912\ldots\).

Modified Answer
\[1 + \sqrt{\frac{537+\sqrt{262769}}{2}}\]
≈ 23.9

Modified Question
Two men and two women stand equally spaced around a circle in random order. What is the probability that every man stands diametrically opposite a woman?

Modified Solution

  1. Label the 4 seats 1–4 around the circle; opposites are (1,3) and (2,4). Total seatings=4!=24.
  2. For both men to be opposite women they must occupy one opposite pair: choose the pair in 2 ways, arrange the 2 men in it in 2! ways, and place the 2 women in the remaining seats in 2! ways, giving 2 · 2 · 2 = 8 favorable.
  3. Thus P = \(\frac{8}{24} = \frac{1}{3}\).

Modified Answer
\[\frac{1}{3}\]

Modified Question
Define f(x)=|| x| - \tfrac{1}{2}| and g(x)=|| x| - \tfrac{1}{4}|. Find the number of intersections of the graphs of \[y=4 g(f(\sin(2 \pi x))) \quad\text{ and }\quad x=0.\] (Fix x=0 instead of the original expression.)

Modified Solution

  1. Set \( x = 0 \). Then \( y = 4g(f(\sin(2\pi \cdot 0))) = 4g(f(0)) \).
  2. Compute \( f(0) = \left| \left| 0 \right| - \frac{1}{2} \right| = \left| -\frac{1}{2} \right| = \frac{1}{2} \).
  3. Compute \( g\left(\frac{1}{2}\right) = \left| \left| \frac{1}{2} \right| - \frac{1}{4} \right| = \left| \frac{1}{4} \right| = \frac{1}{4} \).
  4. Thus \( y = 4 \cdot \frac{1}{4} = 1 \).
  5. Therefore there is exactly one intersection point at \( (0, 1) \).

Modified Question
Let p be the least prime number for which there exists a positive integer n such that n^{4}+1 is divisible by p^{2}. Find the least positive integer m such that m^{4}+1 is divisible by p^{2}. Add condition: Only consider n and m such that m=n.

Modified Solution

  1. Since \( n^4 \equiv -1 \mod p^2 \) forces \( p \equiv 1 \mod 8 \), the least such \( p \) is 17.
  2. Mod 17, we have \( 2^4 = 16 \equiv -1 \), and \( f(x) = x^4 + 1 \) has \( f'(2) = 4 \cdot 2^3 \not\equiv 0 \mod 17 \), so by Hensel's lemma, there is a unique lift modulo \( 17^2 \): \( x = 2 + 17k \).
  3. Expanding \((2+17k)^4+1 \equiv 17(1+32k) \equiv 0 \mod 289 \) gives \( k \equiv 9 \mod 17 \), so the least positive solution is \( n = m = 2 + 17 \cdot 9 = 155 \).

Modified Answer
\[155\]

Modified Question
Among the 900 residents of Aimeville, there are 195 who own a diamond ring, 367 who own a set of golf clubs, and 562 who own a garden spade. In addition, each of the 900 residents owns a bag of candy hearts. Suppose the 234 residents who own exactly three of these things also own a candy store, which means all four items. Find the number of residents who own all four of these things.

Modified Solution

  1. Let \( n_k \) be the number of residents owning exactly \( k \) items.
  2. Modified gives \( n_2 = 437 \) and those 234 with exactly three now own all four, so \( n_3 = 0 \) and let \( x = n_4 \).
  3. From \( n_1 + n_2 + n_3 + n_4 = 900 \), \( n_1 = 900 - 437 - x \).
  4. Also the total item‑count is \( 195 + 367 + 562 + 900 = 2024 \), so \( n_1 + 2 \cdot 437 + 3 \cdot 0 + 4x = 2024 \).
  5. Substituting \( n_1 \) yields: \[(900 - 437 - x) + 874 + 4x = 2024 \implies 3x = 687 \implies x = 229\]

Modified Answer

Modified Question
Let p(x) be a polynomial of degree 5 such that \[p(n) = \frac{n}{n^2 - 1}\] for n = 2, 3, 4, \dots, 7. Find p(8). Added condition: Suppose p(x) is actually a polynomial of degree 1 that fits the given values for n=2 and n=3. Find p(8).

Modified Solution

  1. \(m = \frac{\frac{3}{8} - \frac{2}{3}}{3 - 2} = \frac{9}{24} - \frac{16}{24} = -\frac{7}{24}\)
  2. Line: \(p(x) = mx + b\)
  3. Use \(p(2) = \frac{2}{3}\): \[-\frac{7}{24} \cdot 2 + b = \frac{2}{3} \Rightarrow b = \frac{2}{3} + \frac{7}{12} = \frac{5}{4}\]
  4. Then \[p(8) = -\frac{7}{24} \cdot 8 + \frac{5}{4} = -\frac{56}{24} + \frac{5}{4} = -\frac{7}{3} + \frac{5}{4} = -\frac{28}{12} + \frac{15}{12} = -\frac{13}{12}\]

Modified Answer
\[-\frac{13}{12}\]

Modified Question
Let \[z = 2 + \sqrt{2} - (3 + 3 \sqrt{2})i,\] and let \[c = 2 - 3i.\] Let w be the result when z is rotated around c by \[\pi\] radians counter-clockwise. Find w.

Modified Solution

  1. \(w = c + (z - c)e^{i\pi} = c - (z - c)\)
  2. Compute \(z - c = (2 + \sqrt{2} - 2) + (-(3 + 3\sqrt{2}) + 3)i = \sqrt{2} - 3\sqrt{2}i = \sqrt{2}(1 - 3i)\)
  3. Thus \(w = 2 - 3i - \sqrt{2}(1 - 3i) = (2 - \sqrt{2}) + (3\sqrt{2} - 3)i\)

Modified Answer
\[(2 - \sqrt{2}) + (3\sqrt{2} - 3)i\]

Modified Question
Denali and Nate work for a dog walking business and are paid for each dog they walk. Denali is responsible for 16 dogs and Nate is responsible for 20 dogs. If Denali's pay doubles when she takes on x more dogs from Nate, find x.

Modified Solution

  1. Let p be the pay per dog.
  2. Initially Denali earns 16p.
  3. After taking x dogs, she earns (16+x)p, which doubles her pay: (16+x)p = 2 · 16p = 32p.
  4. Hence 16 + x = 32 and x = 16.

Modified Answer
\[16\]

Modified Question
Find the number of integer values of k in the closed interval [-500,500] for which the equation \[\log(kx) = 2 \log(x+2)\] has exactly one real solution. Assume \[k\geq 0\].

Modified Solution

  1. Domain requires \(k>0\) and \(x>0\).
  2. Rewrite \(\log(kx)=2\log(x+2)\) as \(kx=(x+2)^2 \Rightarrow x^2+(4-k)x+4=0\).
  3. Discriminant \(D=(k-4)^2-16=k(k-8)\).
  4. Exactly one real solution with \(x>0\) occurs only when \(D=0\) and root>0.
  5. \(D=0\Rightarrow k=0\) or \(8\), but \(k>0\Rightarrow k=8\), giving double root \(x=\frac{k-4}{2}=2>0\).

Modified Answer

Modified Question
Let x, y, and z be positive real numbers such that xyz = 2. Find the minimum value of \[x^4 + 4y^2 + 4z^4\] given that x = 1.

Modified Solution

  1. Since \(x = 1\) and \(xyz = 2\), we have \(yz = 2\), so \(y = \frac{2}{z}\).
  2. Substitute into the target: \(f(z) = 1^4 + 4y^2 + 4z^4 = 1 + 4\left(\frac{2}{z}\right)^2 + 4z^4 = 1 + \frac{16}{z^2} + 4z^4\).
  3. Compute \(f'(z) = -\frac{32}{z^3} + 16z^3\) and set to zero: \(16z^3 = \frac{32}{z^3} \Rightarrow z^6 = 2 \Rightarrow z = 2^{1/6}\). Then \(y = \frac{2}{z} = 2^{5/6}\).
  4. The minimum is \(f(2^{1/6}) = 1 + \frac{16}{2^{1/3}} + 4 \times 2^{2/3} = 1 + 8 \times 2^{2/3} + 4 \times 2^{2/3} = 1 + 12 \times 2^{2/3}\).

Modified Answer
\[1 + 12 \cdot 2^{2/3}\]

Modified Question
Find all real numbers x such that \[\frac{x^2 + 2x + 1}{x^2 - 1} = 2\] given that x \neq 1.

Modified Solution

  1. Since \(x \neq 1\), we can multiply both sides by \(x^2 - 1\): \(x^2 + 2x + 1 = 2(x^2 - 1)\).
  2. Expand and simplify: \(x^2 + 2x + 1 = 2x^2 - 2 \Rightarrow -x^2 + 2x + 3 = 0\).
  3. Multiply by -1: \(x^2 - 2x - 3 = 0\).
  4. Factor: \((x - 3)(x + 1) = 0\).
  5. Solutions: \(x = 3\) or \(x = -1\).

Modified Answer
\[x = 3 \text{ or } x = -1\]

Modified Question
Find the sum of all positive integers n such that \(n^2 + n + 1\) is divisible by 7.

Modified Solution

  1. Note that \(n^2 + n + 1 = (n + 1)^2\).
  2. We need \((n + 1)^2 \equiv 0 \pmod{7}\).
  3. This means \(n + 1 \equiv 0 \pmod{7}\).
  4. So \(n \equiv 6 \pmod{7}\).
  5. The positive integers satisfying this are 6, 13, 20, ..., 97.
  6. Sum = \(\frac{14}{2}(6 + 97) = 721\).

Modified Answer
\[721\]

Modified Question
Let f(x) = x^3 - 3x^2 + 2x. Find all real numbers x such that f(x) = 0 and x > 1.

Modified Solution

  1. Factor: \(f(x) = x(x^2 - 3x + 2) = x(x - 1)(x - 2)\).
  2. Set \(f(x) = 0\): \(x(x - 1)(x - 2) = 0\).
  3. Solutions: \(x = 0, 1, 2\).
  4. Only \(x = 2\) satisfies \(x > 1\).

Modified Answer
\[x = 2\]

Modified Question
Find the sum of all positive integers n such that \(n^2 + n + 1\) is divisible by 13.

Modified Solution

  1. We need \(n^2 + n + 1 \equiv 0 \pmod{13}\).
  2. Complete the square: \((n + 7)^2 \equiv 48 \pmod{13}\).
  3. Since 48 ≡ 9 mod 13, we need \((n + 7)^2 \equiv 9 \pmod{13}\).
  4. Thus \(n + 7 \equiv \pm 3 \pmod{13}\).
  5. Solutions: \(n \equiv 9\) or \(n \equiv 3 \pmod{13}\).
  6. Sum = 3 + 9 + 16 + 22 + 29 + 35 = 114.

Modified Answer
\[114\]
